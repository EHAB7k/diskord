## ℹ️ What is diskord?

diskord is a fast, easy-to-use tool to help you manage storage on your Windows computer. It gives you simple text-based views of your disks and files. You do not need to know technical commands to use it.

This app was built for Arch Linux with Hyprland, but it works well on Windows for basic storage management. It helps you see your disk use, clean up space, and organize files without confusing menus.

## 🔧 How to Use diskord

Here’s how to do common tasks inside diskord once it is running.

### Checking Your Storage

- When diskord opens, it displays all your drives with how much space is used and free.
- Use arrow keys to move through your drives.
- Press **Enter** to view the files and folders on a selected drive.

### Finding Large Files

- Press the shortcut key for "find large files." This key is usually shown on the screen or in the help menu.
- Diskord lists files by size, starting with the largest.
- Use arrow keys to scroll through the list.
- Select any file and press **Delete** to remove it if you want to free space.

### Organizing Files and Folders

- Use keyboard keys to sort files by name, date, or size.
- Enter folders by pressing **Enter**.
- Go back by pressing **Backspace** or the key shown for "up."

**Q: Do I need to install any other software?**  
A: No. diskord runs on its own without extra programs.

**Q: Can I use diskord without a keyboard?**  
A: Diskord relies mostly on keyboard controls. A mouse is not required.

**Q: How often should I check disk space?**
